This mod adds Jaffa Cakes (brownies, cream rolls, ice creams and more) into the game. I tried to make a not so easy craft chain - this mod has been mainly created to add extra content which could be used in factory designing and building. It also adds a bunch of [b]fruit trees[/b] that bear fruits (apple, cocoa, vanilla, lemon, orange, plum) and plants (coffee, strawberry, onion, etc).

[size=6]Installation[/size]
0) if you don't have forge, get it - [url="http://www.minecraftforge.net/wiki/Installation/Universal"]http://www.minecraft...ation/Universal[/url]
1) copy jar files (mod and core) into your "minecraft" directory (your minecraft dir on windows is usually found in "c:\Users\<Your user name>\AppData\Roaming\.minecraft\", jaffas and core jar should be in mods directory)

Many things can be turned off in config files. Also in case of ID conflicts you can tell the mod to start assigning IDs from custom numbers.
The mod is checking if there is a newer version available. However you can disable printing of update messages.
For more info check wiki.
